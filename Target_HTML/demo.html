1. Primeros pasos con AsciiDoc

Marcado en línea

  • los subrayados alrededor de una frase ponen énfasis

  • los asteriscos alrededor de una frase hacen que el texto sea negrita

  • los asteriscos dobles alrededor de una o más letras en una palabra hacen que esas letras sean negritas

  • los subrayados dobles alrededor de una sub-cadena en una palabra destaca la sub-cadena

  • usar zanahorias alrededor de los personajes para hacerlos superscript

  • usar tildes alrededor de los caracteres para hacerlos subscript

  • para pasar HTML directamente, rodear el texto con triple plus

  • los caracteres pueden escaparse usando un \

    • por ejemplo, puedes escapar una comilla dentro de un texto enfatizado como Here's Johnny!

  • puedes usar con seguridad los caracteres XML reservados como <, > y &, que se escapan al convertir

  • forzar un espacio entre los elementos en línea usando el atributo {sp}

  • mantener el texto junto con un atributo espacial intrínseco no rompedor, {nbsp}

  • manejar palabras con caracteres unicode como en el nombre Gregory Romé

  • reclamar sus derechos de autor ©, marca registrada ® o marca comercial ™

  • seleccione View ▸ Zoom ▸ Reset para restablecer el zoom

Se puede escribir el texto con enlaces en línea, opcionalmente usando un prefijo de enlace explícito. En cualquier caso, el enlace puede tener una cadena de consulta.

Si quieres romper una línea
sólo tienes que terminarla en un signo +
y seguir escribiendo en la siguiente línea.
